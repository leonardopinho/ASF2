log()function 
public function log(... args):void

Função útil que executa traces marcando sua origem. Por exemplo: 

	 package utils
	 {
	 		class Test
	 		{
	 			public function aMethod( ):void
	 			{
	 				log( "hello!" ); // output: [utils::Test].aMethod( hello );
	 			}
	 		}
	 }
Bastante útil para substituir o trace e ter seu painel de console bem marcado. Internamente ela usa o stack de um erro para mapear de onde veio a chamada da função. Adicionalmente a log tenta executar o método console.log via ExternalInterface no Browser
