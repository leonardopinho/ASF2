Constructor Detail
ApplicationViewController()Constructor
public function ApplicationViewController(p_view:*, p_loader:ILoaderFactoryPlugin, p_id:String, p_model:XML = null, p_loaderInfo:LoaderInfo = null)

Construtor. Mesmos parâmetros que o ASF

Parameters
p_view:* — Instância do container principal onde a estrutura da aplicação deve ser criada. Pode ser qualquer objeto desde que seja um DisplayObjectContainer
 
p_loader:ILoaderFactoryPlugin — Uma instância de uma classe que implemente a interface ILoaderFactoryPlugin.
 
p_id:String — Identificação da Aplicação
 
p_model:XML (default = null) — Um XML estático. Você pode optar por usar o XML estático incorporado ao código ao invés de carregar o XML dinamicamente.
 
p_loaderInfo:LoaderInfo (default = null) — LoaderInfo para ser usado como base de onde buscar os FlashVars que servirão como base de parâmetros da aplicação

trackAnalytics()method 
public function trackAnalytics(metricNode:*, ... replaces):String

Executa o trackAnalytics no Plugin de métricas. Exemplo: app.trackAnalytics( app.metrics.asf.sample.home.navigate, nextSection.id ); ou app.trackAnalytics( "navigate", nextSection.id

Parameters

metricNode:* — O Nó XML ou ID do nó com a métrica a ser executada
 
... replaces — Valores para servirem de variáveis e substituírem os %s dentro da métrica

Returns
String — A URI da métrica aplicada ou vazi no caso de métrica inválida
