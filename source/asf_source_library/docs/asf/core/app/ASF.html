Classpublic class ASF
InheritanceASF ApplicationViewController AbstractViewController flash.events.EventDispatcher

Classe principal do Framework. Exemplo de utilização:

app = new ASF( new Sprite( ), root.loaderInfo, new LoaderMaxFactoryPlugin( ) ); app.addEventListener( DependenciesProgressEvent.LOAD_START, onLoadStart ); app.addEventListener( DependenciesProgressEvent.LOAD_PROGRESS, onLoadProgress ); app.addEventListener( DependenciesProgressEvent.LOAD_COMPLETE, onLoadComplete ); app.params.defaults.config = "../xml/application.xml"; app.loadModel( new URLRequest( app.params.config ) ); addChild( app.view );

Constructor Detail
ASF()Constructor
public function ASF(p_view:*, p_loaderInfo:LoaderInfo, p_loader:ILoaderFactoryPlugin, p_id:String, p_model:XML = null)

Construtor do Framework. Esta classe representa a Applicação principal do projeto. Basicamente, esta é a representação funcional do arquivo XML de configurações

Parameters
p_view:* — Instância do container principal onde a estrutura da aplicação deve ser criada. Pode ser qualquer objeto desde que seja um DisplayObjectContainer
 
p_loaderInfo:LoaderInfo — LoaderInfo para ser usado como base de onde buscar os FlashVars que servirão como base de parâmetros da aplicação
 
p_loader:ILoaderFactoryPlugin — Uma instância de uma classe que implemente a interface ILoaderFactoryPlugin.
 
p_id:String — Identificação da Aplicação
 
p_model:XML (default = null) — Um XML estático. Você pode optar por usar o XML estático incorporado ao código ao invés de carregar o XML dinamicamente.
