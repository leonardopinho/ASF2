interpolate()method 
public function interpolate(p_startValue:Number = 0, p_endValue:Number = 1, p_time:uint = 333, p_easingTransition:Function = null, delay:uint = 0, type:String):ISequence

Cria uma interpolação numérica simples entre dois números.

Parameters

p_startValue:Number (default = 0) — Valor inicial
 
p_endValue:Number (default = 1) — Valor final
 
p_time:uint (default = 333) — Duração em millisegundos
 
p_easingTransition:Function (default = null) — Função de easing. Ex.: Qualquer uma do Robert Penner
 
delay:uint (default = 0) — Tempo de espera antes de iniciar em millisegundos
 
type:String (default = NaN) — Tipo: 'enterFrame' usa a velocidade do framerate do SWF ou 'interval' usa um setInterval de 1000/30 (30 fps)

Returns
